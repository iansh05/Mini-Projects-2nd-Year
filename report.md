### 1. Introduction to Project

#### 1.1 Project Overview
The "Campus Music Fest Website" is a static HTML page promoting a campus music festival. It provides event details, lineup, schedule, tickets, and contact info, built with HTML and CSS.

#### 1.2 Objectives
- Create a responsive website for event promotion.
- Showcase web development skills in layout and styling.
- Enable ticket inquiries.

#### 1.3 Scope
Focuses on a single HTML file with embedded CSS, including hero, lineup, schedule, tickets, and contact sections. Future JS integration possible.

- **Schedule Section**: Table listing event schedule by day, time, artist, and stage.
